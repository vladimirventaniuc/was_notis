Post-test Questions Answers

The questions are subjective and open. Their purpose is to identify what can be done to improve the user experience.

Test Observations

Obs 1: For some users it was difficult to use the web application especially if they didn't use applications like Google Maps in the past.

Obs 2: Web application is easy to use for the average person that had some interaction with a web browser before.

Obs 3: Design can be improved by making it to look better without adding extra complexity in it.

Potential Improvements

Some data used in the application is not updated. We can update the informations and add new new information for each entity.

Web application will not work if the external Google Maps API is not available. We can integrate multiple APIs that offer interactive maps.
